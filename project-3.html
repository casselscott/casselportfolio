Water Quality Portability

Features


Predicting Water Potability

Information about the features or factors contributing to the quality of water, basically answering the question if the water is portable or not. From the data prediction,the knowledge of the results derived can help reduce risk in terms of spread of harmful bacterias and diseases.

Machine Learning

Three different machine learning model algorithm was used to determine the suitable model that meets the problem statement requirement.

Data Insights

Identifying patterns and trends, correlations and anomalies within the dataset. Displays easy to understand visualisations using visual representations charts such as Scatter plot, Bar, Line chart

User Friendly Interface

A simple user-interface that displays the exploratory data analysis as well as the visualisations for users.
